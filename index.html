GIT - система контролю версій

1. Переходимо в командну строку або в TERMINAL вводимо команду git init після ініцілізації з'являється прихована папка .git

2. Налаштовуємо систему GIT, потрібно зареєструвати ім'я та емейл для проекту. Прописуємо конфігураційні команди локально або глобально: для імені - git config --local user.name "Alex", для емейла - git config --local user.email example@gmail.com. Відповідно для глобального застосовується команда git config --global user.name "Alex" тоді ім'я буде застосовано до всіх проектів на комп'ютері...

3. Перевірка статусу репозиторія git status

4. Добавлення всіх файлів в індекс git add -A, по одному git add index.html, або за розширенням git add *.html

5. Виконуємо збереження контрольної точки git commit -a -m"first commit"

6. Виконуємо збереження контрольної точки git commit -a -m"first commit"

7. Виконуємо збереження контрольної точки git commit -a -m"first commit"
